# FinPlanner - Planejador Financeiro

## 📝 Descrição

O FinPlanner é uma aplicação web para gerenciamento financeiro pessoal que permite controlar despesas de diferentes tipos: à vista, cartão de crédito e recorrentes.

## 🚀 Funcionalidades

- Cadastro de despesas com categorização
- Suporte a diferentes tipos de pagamento:
  - À Vista
  - Cartão de Crédito (com parcelamento)
  - Despesas Recorrentes
- Visualização de gastos por período
- Interface responsiva e amigável
- Suporte a PWA (Progressive Web App)

## 💻 Tecnologias

- HTML5
- CSS3
- JavaScript
- PWA (Progressive Web App)

## 🎯 Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador
3. Comece a registrar suas despesas!

## 📱 PWA

A aplicação pode ser instalada como um aplicativo em dispositivos móveis através do recurso PWA.

## 🔒 Privacidade

Todos os dados são armazenados localmente no seu dispositivo.
